# aPersona Identity Manager — Multi-Tenant Platform (Release)

> Pre-built release artifacts. Do NOT edit files in this repo directly.
> This repo is automatically populated by the CI/CD pipeline.

## Quick Start

1. Clone this repo
2. Edit `tenants-config.json` with your deployment settings
3. Run `./install-multi-tenants.sh`

## Structure

```
├── amfa-service-multi-tenants/      # Pre-built identity service
├── amfa-admin-portal-multi-tenants/ # Pre-built admin portal
├── install-multi-tenants.sh         # Main installer
├── uninstall.sh                     # Uninstaller
├── tenants-config.json              # Configuration template
└── VERSION                          # Current version
```
